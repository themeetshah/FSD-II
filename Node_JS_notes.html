Node.js

  • It runs single-threaded, non-blocking, asynchronus programming

  • REPL: Read-Eval-Print Loop

    • Read: The REPL reads the user input (typically a line of code).
    • Eval: It evaluates the code you've entered, executing it or computing the result.
    • Print: It prints the result of the evaluation, so you can see what happened.
    • Loop: It repeats the process, allowing you to enter more code.
      SIMPLE Expression:
    • > 1+3
    • > 1+(2*3)-4
    • > var x = 10
    • > var y = 20
    • > x = y
    • > console.log('Hello World')
    • > repl.repl.ignoreUndefined = true
      Underscore variable:
    • Used to get last result
    • > var x = 10;
    • > var y = 20;
    • > x + y
    • > var sum = _
    • > console.log(sum)
      Editor mode:
    • > .editor
      Commands:
    • >.save filename
    • >.load filename
