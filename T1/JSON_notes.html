JSON: JavaScript Object Notation

    • It is a text format for storing & transporting data.
    • easy to read, write & understand as it is 'self-describing'.
    • language independent
    • Syntax is similar to JavaScript object literal syntax.
    • Represented in key/value pair
    •         Syntax:
            {
                key : value,
                key : value
            }
    • Key can or can't be in quotes

  • JSON supports following datatypes:

    • String: {"Name" : "xyz"}
    • Number: {"age" : 30}
    • Object: {employee : { "FirstName": "xyz", "LastName": "abc", "id": 'st1' }}
    • Array: {"nums" : [1,2,3,4]}
    • Boolean: { "FirstName": true, "LastName": false }
    • Null: {"value" : null}

  • parse() method

    • Syntax: JSON.parse(string)
    • It is a built-in function for converting JSON strings into JS objects
    • Example: 
                              let obj = `{
                            "var1" : "LJ",
                            "var2" : "University"
                        }`
                        var obj1 = JSON.parse(obj)
                        console.log(obj1.var1)
                        console.log(obj1.var1+" "+obj1.var2)

  • stringify() method

    • Syntax: JSON.stringify(object)
    • It is a built-in function for converting JSON object into string
    • Example: 
                              let obj = `{
                            "var1" : "LJ",
                            "var2" : "University"
                        }`
                        var obj1 = JSON.stringify(obj)
                        console.log(obj1.var1)
    • This won't work as now obj1 has become a STRING: console.log(obj1.var1+" "+obj1.var2)
