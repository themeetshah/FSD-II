fs module

  • works on 2 modes - synchronous and asynchronous
  • operations: read,write,update,delete,renam

  • Synchronous functions:

    • readFileSync(filename,"utf-8") (returns data in unicode format) (so utf-8 to get data in readable form)
    • OR ps.readFileSync("filename")
    • writefIleSync(filename,"data") (creates new file if does not exists else overwrites)
    • ps.toString()
    • appendFileSync(filename,"data") (creates new file if does not exist)(appends at the end of the file)
    • unlinkSync(filename) -- (del the file)
    • renameSync("OldFileName","NewFileName")

    • For directory

      • - mkdirSync(name) --makes new directory
      • - rmdirSync(name)
